# 作用域


+ 使用键值对(key=value)存储数据的地方



## 四大作用域


1. page: 页面级别, 基本不再使用.
2. request: 请求级别, 在同一次的请求中有效, 即只有同一次请求才能访问到request作用域中的数据.
3. session: 会话级别, 在同一次的会话中有效, 即只有同一个会话才能访问到session作用域中的数据.
4. application: 应用程序级别, 在同一个应用程序(同一个Web工程)中有效,即所有使用此应用程序的user都能访问到application中的数据.



## 各个作用域的使用


+ session作用域只允许同一个会话访问,因此可以用来存储user的信息及一些个人,私密的信息, 只允许本人查看
+ application作用域允许所有人访问, 可以用来发布一些公告, 公告资源等等
