## 绝对路径


+ http://ip:port/Web工程/Web资源路径
+ HTML: : 使得当前页面的所有连接都以此为基础进行资源访问
+ thyemleaf: th:herf=@{xxx}: 使得大括号中的路径从本Web工程开始访问资源



## 相对路径


+ ./  当前文件所在的目录.
+ ../ 当前文件所在的父级目录, 可以多次使用, 不断向外延伸.



### 相对路径中的" / "的解析


+  客户端解析" / " 

```plain
  http://ip:port/, 只能解析到端口号, 即解析到一台具体的服务器
```

 

+  服务器解析" / " 

```plain
  http://ip:port/Web工程/, 能解析到具体工程, 即解析到本服务器的一个具体的Web工程
```
