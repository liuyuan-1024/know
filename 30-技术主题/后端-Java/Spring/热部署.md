# Spring热部署
## 引入热部署依赖
在需要热部署的模块中引入热部署依赖

```xml
<!-- 热部署 -->
<dependency>
    <groupId>org.springframework.boot</groupId>
    <artifactId>spring-boot-devtools</artifactId>
    <!-- 打包时不将devtools打包到jar包中 -->
    <scope>runtime</scope>
    <!-- 防止依赖传递到其他模块 -->
    <optional>true</optional>
</dependency>
```

## 添加插件到pom.xml中
将插件添加到父工程或子工程的pom.xml中



```xml
<build>
    <plugins>
        <plugin>
            <groupId>org.springframework.boot</groupId>
            <artifactId>spring-boot-maven-plugin</artifactId>
            <version>${spring.version}</version>
            <configuration>
                <fork>true</fork>
                <addResources>true</addResources>
            </configuration>
        </plugin>
    </plugins>
</build>
```
